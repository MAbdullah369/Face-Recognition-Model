# Face Recognition Model

A comprehensive face recognition system built with Python, featuring user registration, authentication, and management capabilities through an intuitive web interface.

## Features
- User Registration with facial data capture
- Real-time Face Authentication
- User Management interface
- Web-based GUI using Gradio
- Local data storage for privacy
- High-accuracy face recognition

## Installation

### Prerequisites
- Python 3.8 or higher
- Webcam
- Internet connection (for package installation)
